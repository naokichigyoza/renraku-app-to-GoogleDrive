# れんらくアプリ自動保存

幼稚園や学校で、[れんらくアプリ（BusCatch）](https://www.buscatch.com/renraku-app/) を使ってお知らせを受け取っている方向けのツールです。

れんらくアプリから届く通知メールをきっかけに、お知らせ本文と添付ファイル（PDFなど）を、Google Driveに自動保存します。

これで「後で確認しようと思って忘れていた・・・！」と困ることが無くなります。

このツールは、れんらくアプリ公式のサービスではありません。個人が作成した非公式ツールです。

## こんな人向けです

- 園や学校からのお知らせを れんらくアプリ で受け取っている
- れんらくアプリの通知を Gmail で受け取れる
- お知らせのPDFなどを、自動で保存しておきたい
- 家族であとから見返せるように、Google Driveにまとめたい

## 保存されるもの

初期設定のまま使うと、Google Driveの「マイドライブ」に `renraku-app` というフォルダが作られます。

その中に、メールごとにフォルダが作られます。

```text
renraku-app/
  20260707_1911_7月おやじの会/
    本文.txt
    6184541_1.pdf
```

保存されるのは次の2つです。

- お知らせのタイトルと通知メールの本文
- お知らせページにログインして取得できる添付ファイル（PDFなど）

## 使い方

### 1. れんらくアプリの通知を Gmail で受け取る

まず、れんらくアプリからの通知メールを Gmail で受け取れるようにしてください。

Gmailに届いたメールの送信元が `@buscatch.net` であれば対象になります。

メール本文に、次のようなURLが含まれていれば対象になります。

```text
https://buscatch.net/mobile/xxxxx/open_confirm_mail/open/?m=...&u=...&s=...
```

### 2. Google Apps Script を開く

<https://script.google.com> を開いて、「新しいプロジェクト」を作成します。

### 3. コードを貼り付ける

左側の `コード.gs` を開き、中身をすべて削除します。

そこに、このリポジトリの [Code.gs](./Code.gs) の中身を貼り付けて保存します。

### 4. ログイン用パスワードを設定する

れんらくアプリのお知らせページを開くには、共通のパスワード（園から配布されているもの）が必要です。

このパスワードは `Code.gs` に直接書かず、Apps Script の「スクリプト プロパティ」に設定します。こうすることで、GitHubなどにパスワードが残る心配がありません。

1. Apps Script の画面左側にある、歯車アイコン（プロジェクトの設定）をクリックします。
2. 下の方にある「スクリプト プロパティ」までスクロールし、「スクリプト プロパティを追加」を押します。
3. プロパティ名に `LOGIN_PASSWORD`、値に れんらくアプリのパスワードを入力し、「スクリプト プロパティを保存」を押します。

### 5. まず一度実行する

上部の関数選択で `今すぐ保存する` を選び、「実行」を押します。

初回だけ、Googleアカウントの権限確認が表示されます。

これは、このスクリプトが Gmail と Google Drive を使うためです。

途中で「このアプリは Google で確認されていません」または「Google hasn't verified this app」と表示されることがあります。自分で作成したApps Scriptでも、Googleによるアプリ確認を受けていないため、この画面が出る場合があります。

その場合は、次のように進めます。

1. 「詳細」を押す
2. 「無題のプロジェクト（安全ではないページ）に移動」を押す
3. 権限の内容が表示されたら、チェックボックスにチェックを入れる
4. 「続行」または「許可」を押す

実行が終わったら、Google Driveのマイドライブに `renraku-app` フォルダができているか確認してください。

### 6. 自動実行を設定する

問題なく保存できたら、関数選択で `初回に1回だけ実行する_自動保存を開始` を選び、「実行」を押します。

これで1時間ごとに自動で確認し、新しいメールがあれば保存します。

## 設定を変えたい場合

そのまま使う場合は、ここは読まなくても大丈夫です。

保存先フォルダ名などを変えたい場合は、`Code.gs` の先頭にある `CONFIG` を編集してください。

```js
const CONFIG = {
  DRIVE_FOLDER_NAME: 'renraku-app',
  DRIVE_FOLDER_ID: '',
  GMAIL_QUERY: 'from:(@buscatch.net)',
  PROCESSED_LABEL: 'renraku-app-to-GoogleDrive/saved',
  FAILED_LABEL: 'renraku-app-to-GoogleDrive/failed',
  TIMEZONE: 'Asia/Tokyo',
  MAX_THREADS_PER_RUN: 50,
};
```

それぞれの項目の説明は、`Code.gs` のコメントに書いてあります。

パスワードだけは `CONFIG` には含まれておらず、スクリプト プロパティの `LOGIN_PASSWORD` から読み込みます。

## うまくいかないとき

| 困ったこと | 確認すること |
|---|---|
| 何も保存されない | Gmailで `from:(@buscatch.net)` と検索して、対象メールがあるか確認してください。 |
| 「LOGIN_PASSWORD が設定されていません」と出る | スクリプト プロパティに `LOGIN_PASSWORD` を設定してください（手順4）。 |
| 「パスワード認証に失敗しました」と出る | スクリプト プロパティの `LOGIN_PASSWORD` の値が正しいか確認してください。園でパスワードが変更された場合は、値を更新してください。 |
| 添付ファイルが保存されない | お知らせにファイルが添付されていない可能性があります。本文.txt の内容を確認してください。 |
| 同じメールがもう一度保存された | Gmailの処理済みラベルを外すと、再度保存されます。 |
| 保存先を変えたい | `Code.gs` 先頭の `DRIVE_FOLDER_NAME` を変更してください。 |

## 注意

- このツールは、本文に `https://buscatch.net/mobile/.../open_confirm_mail/open/...` のリンクが含まれるメールを対象にしています。
- れんらくアプリ公式とは関係のない、個人制作の非公式ツールです。
- ログイン用パスワードは、園から配布された共通パスワードをそのまま使う想定です。第三者と共有しないよう注意してください。
